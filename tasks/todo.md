# AI Educational Website

## Plan

Single `index.html` file with inline CSS and minimal JS.

### Structure
- [x] Hero section — bold headline, subtext, CTA button
- [x] "What You'll Learn" — 3-column feature cards with placeholder images
- [x] Course modules section — numbered list with short descriptions
- [x] Testimonials — 2–3 quote cards
- [x] Footer — nav links + copyright

### Design
- Color palette: off-white background, near-black text, one accent (electric indigo `#6366f1`)
- Typography: system sans-serif stack (or Google Font: Inter via CDN)
- Layout: CSS Grid + Flexbox, generous whitespace
- Placeholder images: `https://placehold.co/` service
- No frameworks, no build step — pure HTML/CSS

### Notes
- Keep it under ~300 lines
- No JavaScript interactivity needed (maybe a subtle scroll nav active state)

## Review

**What was built:** `index.html` — a single-file AI educational website for a fictional brand "Meridian."

**Sections delivered:**
- Sticky nav with blur backdrop
- Hero with headline, stats row, and placeholder image
- 3-column feature cards (Foundations / Labs / Ethics)
- 8-module curriculum list with sticky visual panel
- 3-column dark testimonials
- Indigo CTA banner
- Footer

**Design choices:** Inter font, `#6366f1` indigo accent, off-white background, generous whitespace, subtle hover lifts — intentionally minimal and agency-clean.

**Follow-up ideas (not implemented):**
- Add a real favicon
- Animate the hero headline on load
- Add a mobile hamburger menu for the nav links
